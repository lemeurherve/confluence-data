Cheat sheet

To test connectivity to localhost (via unix docmain socket)

ldapwhoami -x -D 'cn=kohsuke,ou=people,dc=jenkins-ci,dc=org' -W

To test connectivity to another host (note that LDAP doesn't accept connection from everywhere)

ldapwhoami -x -D 'cn=kohsuke,ou=people,dc=jenkins-ci,dc=org' -W -H ldaps://ldap.jenkins.io/

To backup,

$ slapcat | gzip

To restart backup from data.ldif.gz

$ /etc/init.d/slapd stop
$ cd /var/lib
$ mv ldap ldap.backup
$ mkdir /var/lib/ldap
$ gunzip -c data.ldif.gz | slapadd
$ chown -R openldap:openldap /var/lib/ldap
$ /etc/init.d/slapd start
