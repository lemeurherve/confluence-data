Jenkins Features Controlled with System Properties

Jenkinsにはいくつかの"隠し"機能があり、システムプロパティで有効にできます。システムプロパティは、-Dproperty=valueをjavaコマンドラインに渡してJenkinsを開始することで設定されます。これらの引数は、全て引数-jarの前で渡すことに注意してください。そうではない場合、それらは無視されます。

例:

java -Dhudson.footerURL=http://example.org -jar jenkins.war

以下は、プロパティとそれらが導入されたJenkinsのバージョンの一覧です。

  • プロパティ - Javaプロパティ名
  • デフォルト - 明示的に指定しない場合のデフォルト値
  • バージョン - このプロパティが導入されたJenkinsのバージョン
  • 備考 - その他の覚書
