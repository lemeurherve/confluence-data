Jenkins Pluginの翻訳状況 (2011年3月29日現在)

以下のプラグインが日本語化されています。

  • Phing Plugin PHP用のプロジェクトビルドツールであるPhingを使用するためのプラグインです。
  • Copy Artifact Plugin 他のプロジェクトの成果物をコピーするビルドステップを提供するプラグインです。
  • Setenv Plugin プロジェクト単位で環境変数を設定できるプラグインです。
  • Cobertura Plugin Coberturaが生成した情報をもとに、カバレッジ情報を表示するプラグインです。
  • Warnings Plugin コンソールログに出力されたコンパイラーの警告を集計・表示するプラグインです。
  • Checkstyle Plugin Checkstyleの結果を集計・表示するプラグインです。
  • DRY Plugin PMDの結果をもとに、重複したコードを集計・表示するプラグインです。
  • FindBugs Plugin FindBugsの結果を集計・表示するプラグインです。
  • Task Scanner Plugin コード中にある TODO 等のキーワードを抽出し、集計・表示するプラグインです。
  • PMD Plugin PMDの結果を集計・表示するプラグインです。
  • Analysis Collector Plugin 上記6個のプラグインの結果を1つの推移グラフにまとめるプラグインです。
  • JIRA Plugin ATLASSIAN社のBTSであるJIRAと連携するためのプラグインです。
  • Mantis Plugin オープンソースのBTS MantisBTと連携するためのプラグインです。
  • Promoted Builds Simple Plugin Promoted Builds Pluginの簡易版です。ジョブに手動で"リリース版"等のラベルを設定できるプラグインです。
  • Change Log History Plugin 削除されたビルドの変更ログを直近のビルドに継承するためのプラグインです。
  • Dashboard View Plugin ポータルライクな新しいビューを追加するためのプラグインです。
  • Email-ext Plugin Jenkins組込みのE-mail通知の拡張版です。
  • Clover Plugin カバレッジツールCloverのプラグインです。
  • Sidebar-Link Plugin Jenkinsのグローバルメニューまたはプロジェクトメニュー)にカスタムメニューを追加するためのプラグインです。
  • Disk Usage Plugin ジョブのbuildsディレクトリとworkspaceディレクトリのディスク使用量を計測・表示するプラグインです。
