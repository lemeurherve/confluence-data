Jenkins : Hudson windows service fails to start

Created by Unknown User (kohsuke), last modified on Jul 01, 2009

Service Fails to Start

Symptoms

  • Event Viewer, App: "Service cannot be started. The service process could not connect to the service controller"
  • No log files in Hudson directory
    • Repeated attempts to start server may result in 'hudson.err.log' and 'hudson.out.log'
  • hudson.war file is zero length

Solution

  • Copy original 'hudson.war' into 'hudson' directory and restart service
