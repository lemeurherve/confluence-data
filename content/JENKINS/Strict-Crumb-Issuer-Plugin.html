Description

The Strict Crumb Issuer plugin is an extended version of the Default Crumb Issuer embedded in Jenkins core. It provides advanced options of configuration.

It's strongly recommended to use a Crumb Issuer (this one or the embedded one), otherwise your instance will not be protected against CSRF attacks.
