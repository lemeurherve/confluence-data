FindBugs Maven Module Statistics

The FindBugs plug-in results are presented on a summary screen showing the warning statistics by maven or ant module. For each scanned FindBugs file an summary graph shows the priority distribution of the warnings. By clicking on a module link this result is further subdivided into a package statistic per module.
