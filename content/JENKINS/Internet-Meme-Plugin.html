Jenkins : Internet Meme Plugin

Created by Unknown User (harpreet100), last modified on Jan 22, 2014

Plugin Information

View InternetMeme on the plugin site for more information.


Displays a random "internet meme" with a build related message on each build page.Â 

Inspired by the Chuck Norris plugin.

Download & Installation

You can download it directly from the Plugins Update Center.

Usage

  1. Go to the job configuration page and check the "Activate Internet Meme" checkbox in the "Post Build Actions" dropdown
  2. Post build run, a random meme image is displayed on the build page.
