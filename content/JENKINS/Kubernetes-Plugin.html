Jenkins : Kubernetes Plugin

Created by Unknown User (csanchez), last modified by Unknown User (jglick) on Sep 04, 2019

This plugin allows Jenkins agents to be dynamically provisioned on a Kubernetes cluster.

Plugin Information

View Kubernetes on the plugin site for more information.

Older versions of this plugin may not be safe to use. Please review the following warnings before using an older version:

  • CSRF vulnerability and missing permission checks allowed capturing credentials
  • Jenkins controller environment variables accessible
  • Lack of masking of secrets in build log
  • Missing permission check allows listing pod templates
  • Missing permission check allows enumerating credentials IDs
See GitHub for all information.
