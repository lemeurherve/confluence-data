Changelog

Version 1.6 (Jan 16, 2017)

  •   Please read this Blog Post before upgrading
  •   Tombstone release that will migrate configuration and data from previous releases of the plugin.
  • Once you have upgraded to this plugin, you can safely uninstall this plugin after a full re-index of all org folders and multi-branch projects.
  • There will be no more releases of this plugin

Version 1.6-beta-1 (Dec 16, 2016)

  • Once you have upgraded to this plugin, you can safely uninstall this plugin after a full re-index of all org folders and multi-branch projects.
  • There will be one more release of this plugin, i.e. a tombstone for the regular update center
