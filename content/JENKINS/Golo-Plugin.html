Plugin Information

No information for the plugin 'golo-plugin' is available. It may have been removed from distribution.

This plugin let you execute Golo code.

About

This plugin makes it possible to invoke Golo code as build step.

Configuration

To configure available Golo installation on your system, go to Jenkins configuration page, find section 'Golo' and fill the form as shown bellow.

Usage

You can use the plugin to invoke either a Golo script or a inline Golo command.



Changelog

  • Invoke a Golo inline command
