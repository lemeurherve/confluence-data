Jenkins : Centralized Job(Re)Action Plugin

Created by Unknown User (tspengler), last modified by Unknown User (jan_ruzicka) on Aug 13, 2013

Centralized configuration to restart jobs when a pattern occurs in the job console output.

Plugin Information

View Centralized Job(Re)Action on the plugin site for more information.

Intention

To reschedule build in a centralized way when a predefined pattern exist in the job output

Configuration

at http://<yourHudson>/configure

By default this plugin is not active.
