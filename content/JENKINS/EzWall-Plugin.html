EzWall Plugin

Plugin Information

View EzWall on the plugin site for more information.


Add a button to display a fullscreen buildwall on each view.

Configuration

Just activate the plugin and lick on the EzWall button on any view.

Poll interval is configurable through Jenkins configuration page in the EzWall section and modifiable in the settings view when the wall is displayed.
