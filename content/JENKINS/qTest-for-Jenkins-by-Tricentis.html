qTest by Tricentis now integrates seamlessly with Jenkins, helping make your Continuous Integration process more efficient.

When creating builds in Jenkins, your JUnit test results can be sent directly to your qTest Test Case Management software as test runs and test logs. This lets your team review the quality of your build process over time and across different environments, helping you quickly identify problem areas.


About

The qTest for Jenkins add-on will collect JUnit test results from your builds and submit them as Test Runs and Test Logs to qTest. The qTest plugin supports all testing frameworks that are capable of generating JUnit XML format test reports.

After installing, you can set up a post build action and easily configure the connection to your qTest project (shown below).

Major features include:

  • Test Cases, Test Runs and Test logs automatically created in qTest for executed JUnit tests
  • View JUnit test results for build deployments
  • Track JUnit test results across multiple builds
  • Export JUnit test execution results – Build by Build or Test by Test
  • View the submission status of JUnit tests sent to qTest (shown below)
  • Import test cases based of method or class 

About Tricentis

Tricentis is a software testing company founded in 2007 and headquartered in Vienna, Austria. It provides software testing and software quality assurance solutions.
