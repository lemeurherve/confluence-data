Jenkins : Frugal Testing plugin

Created by Unknown User (jakshatdesai), last modified on Jul 08, 2019

This plugin allows you to run load tests on Frugal Testing from Jenkins.


PLUGIN INFORMATION

View Frugal Testing Plugin on the plugin site for more information.


PRESENTATION

Use this plugin to execute automatic load test scripts. Test results are downloaded automatically as JMeter JTL files on your local machine.


PREREQUISITES

Frugal Testing Jenkins plugin requires:

  • Jenkins 2.150.3 or later,
  • A Frugal Testing account. You can create one here if you haven't any.
