Jenkins : Inject environment variable into winrm command

Created by Unknown User (ph2154) on Nov 14, 2018

I cannot inject environment variable into winrm invoke command section

Step-by-step guide

  1. Create environment variable WIN_SERVICE=IOSService in Build Environment
  2. Create WinRM Client in Build
  3. Add "Write-Host APP ${ENV:WIN_SERVICE}" in invoke command
  4. build it
  5. Cannot retrieve the environment variable WIN_SERVICE
