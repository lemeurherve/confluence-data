Jenkins : ICQ Notification Plugin

Created by Unknown User (nolequen), last modified on Apr 14, 2019 

The plugin allows to send Jenkins notifications to ICQ

Plugin Information

View ICQ and MyTeam Notification on the plugin site for more information.

Usage

  • Create your own bot by sending the /newbot command to MegaBot and follow the instructions
  • Specify your bot token in Jenkins system configuration
  • Add a build step and/or a post build action
  • Fill the message and recipients
