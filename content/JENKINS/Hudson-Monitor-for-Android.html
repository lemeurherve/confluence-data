Jenkins : Hudson Monitor for Android

Created by Unknown User (xlv), last modified on Feb 02, 2009

Monitor and display the status of your builds on your Androidâ„¢ phone.
The application will periodically fetch the status of the builds and notify you of new build failures.

Download & Installation

Start the Android Market application on your phone and search for "Hudson" to download the latest release.

Usage

Use the preferences page (Menu -> Settings) to change the RSS feed URL. For now, the entire URL must be provided so the best way to get it is probably to open the server site on your phone and copy/paste one of the feed links at the bottom of the page you want to monitor.

Select an appropriate refresh rate.

Specify the type of notification you want to receive:

  1. Enable/Disable notifications for build failures.
  2. Select the ringtone associated with notifications.
  3. Enable/Disable the vibrate option.

Once the setup is complete, you will see a list of builds with the familiar icons representing their status. Clicking on a build will open the build web page in a browser window.

The build failure notifications visible in the status bar will also open the build web page when selected.
