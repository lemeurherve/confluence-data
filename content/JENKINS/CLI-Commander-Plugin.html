CLI Commander Plugin

Run CLI commands from Jenkins web UI.

What? Why?

While some operations are easy to perform from UI, other are easier to do using CLI commands. Users often decide not to use Jenkins CLI as the setup is not trivial, especially when authentication is required. CLI Commander Plugin give users instant access to CLI features for experiments or occasional use.
