Jenkins : Pipeline Input Step Plugin

Created by Unknown User (jglick), last modified by Unknown User (dnusbaum) on Aug 27, 2019

Plugin Information

View Pipeline: Input Step on the plugin site for more information.

Older versions of this plugin may not be safe to use. Please review the following warnings before using an older version:

  • Users with read access could interact with input step by default
Adds the Pipeline step ‘input’ to wait for human input or approval.

A component of Pipeline Plugin.
