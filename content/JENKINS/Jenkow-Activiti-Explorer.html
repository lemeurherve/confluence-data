Adds the Activiti Explorer and Modeler (web-based BPMN workflow editor) to Jenkow.

The Activiti Explorer is a web application that provides access to the Activiti Engine runtime for all users of the system. It includes task management, process instance inspection, management features and viewing reports based on statistical history data, and comes most importantly with a web-based workflow editor, the Activiti Modeler.

The Jenkow Activiti Explorer plugin encapsulates the Activiti Explorer & Modeler, served out by Jenkins.
