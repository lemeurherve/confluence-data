Jenkins : XP-Dev plugin

Created by Unknown User (ndeloof), last modified by Unknown User (orrc) on May 04, 2015

Plugin Information

View XP-Dev on the plugin site for more information.

This plugin let you configure a web hook on http://xp-dev.com/ hosted repositories to trigger jenkins builds.

Usage

Edit your repository configuration http://xp-dev.com/sc/edit/<id> and set the webhook-url as http://JENKINS/xpdev-webhook
