Jenkins : IPMessenger Plugin

Created by Unknown User (nabedge), last modified on Dec 02, 2015

Plugin Information

View IPMessenger on the plugin site for more information.



This plugin send notification by IPMessenger protocol.

IPMessenger is a simple IM software using UDP.

  • http://ipmsg.org/ (Japanese)
  • http://ipmsg.org/index.html.en (English)

With this plugin, jenkins-ci sends message after build job via IPMessenger protocol.
Enjoy!
