Jenkins : Elasticsearch Query Plugin

Created by Unknown User (mikee805), last modified on Dec 14, 2015

Plugin Information

View Elasticsearch Query on the plugin site for more information.

Fails a build based upon an elasticsearch query

About this plugin

This came from a need to alert based on data stored on in elastic search; in particular logstash data. ELK users who also use Jenkins now have FOSS capabilities similar to that of watcher or splunk.
