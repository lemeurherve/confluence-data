Plugin Information

View OpenShift Client on the plugin site for more information.

The OpenShift Client Plugin aims to provide a readable, concise, comprehensive, and fluent Jenkins Pipeline DSL syntax for rich interactions with an OpenShift API Server. The plugin leverages the OpenShift command line tool (oc) which must be available on the Jenkins nodes executing the Pipeline.
