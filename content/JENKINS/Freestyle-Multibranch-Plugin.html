Freestyle Multibranch Plugin

This is a multibranch project type that applies the same configuration to all branches.

Background

While developing the ideas that became the literate project type, this was one of the initial early steps.

Multi-Branch Project Plugin does something similar but is released. On the other hand it does not integrate with the Branch API plugin.
