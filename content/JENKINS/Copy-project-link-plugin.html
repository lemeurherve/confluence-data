Jenkins : Copy project link plugin

Created by Unknown User (vjuranek), last modified by Unknown User (olivergondza) on May 13, 2015

Plugin Information

View Copy project link on the plugin site for more information.


This plugin adds the "Copy project" link into left side panel in the main project page to facilitate copying of job configuration.
