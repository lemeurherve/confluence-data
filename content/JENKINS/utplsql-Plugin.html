Jenkins : utplsql Plugin

Created by Unknown User (kusemuckl), last modified on Jan 28, 2017

Plugin Information

View utPLSQL on the plugin site for more information.

The utplsql plugin is a tool to parse the output of utplsql to have test reports in Jenkins.

This plugin is for utplsql v2 only. For v3 please use the xunit plugin.
