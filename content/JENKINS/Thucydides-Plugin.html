A plugin for publishing web test reports created by Thucydides.

Overview

The Thucydides plugin collects and archives the test output and summaries created by a Thucydides web test suite. The reports of the latest build are published under http://JENKINS/jobs/YOURJOB/thucydides/. An icon on the job status page links to this URL:
