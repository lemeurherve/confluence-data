Jenkins : Arbitrary Parameters Plugin

Created by Unknown User (louisrli), last modified on Jul 13, 2012

Allows string parameters to be specified at build-time in properties file format

Plugin Information

No information for the plugin 'arbitrary-parameters' is available. It may have been removed from distribution.

Overview

This plugin gives the option for a parameter, known as "Arbitrary Parameters," to be defined through a textbox. This allows any number of parameters to be specified in property values format.

Example

KEY0=MYVALUE
KEY1=My value with spaces
