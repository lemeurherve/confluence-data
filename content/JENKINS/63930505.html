Jenkins : Sonatype CLM (formerly Insight for CI)

Created by Unknown User (mcculls), last modified by Unknown User (bhamail) on Apr 23, 2015

Overview

The Sonatype Insight Plugin for Jenkins is no longer available. The free plugin has been replaced by a new application, Sonatype CLM. It is available for Jenkins and includes a range of new features built around governance and policy management for component usage. This is no longer a free/freemium product, and as such there is no longer a free, self-service trial available. For more information on the latest for Sonatype CLM please visit our Sonatype CLM web page. For additional information on the latest version and installation instructions for Sonatype CLM, as well as our other range of products, please review our support page: support.sonatype.com.
