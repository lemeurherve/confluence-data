Threshold Trend

A trend report is shown on the project page that shows the number of warnings for each build. If the unstable threshold has been set, then the graph uses two colors: red indicates that the number of warnings exceeds the threshold, blue indicates that the number of warnings is within the specified tolerance level.

You can toggle between this graph and the priorities trend by clicking the (show priorities) link.
