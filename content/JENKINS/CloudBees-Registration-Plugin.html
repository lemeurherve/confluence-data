Plugin Information

No information for the plugin 'cloudbees-registration' is available. It may have been removed from distribution.

This plugin builds on top of CloudBees Credentials plugin and provides a concrete implementation and user interface to let users enter the credentials. This is normally not a plugin that you'll choose to install on its own. Rather, it gets installed as a dependency of other plugins.

If you are a plugin developer interested in developing a plugin that accesses CloudBees services, you should depend on this plugin to reuse the same credentials.

Usage

Upon the first installation (or if a cloudbees username/password is not set), then the plugin puts up a banner that asks you to do so:

You can then enter your CloudBees credentials:

The information you entered is managed by the Credentials Plugin. To update/delete it, or to add the 2nd account information, click the "Credentials" link from the left.
