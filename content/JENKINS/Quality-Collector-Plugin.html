No information for the plugin 'quality-collector' is available. It may have been removed from distribution.

The plugin "Quality Collector" is designed to give a (more or less) detailed overview over all gathered test results of different test tools used to test your application/development artefacts. (currently Junit and SoapUI supported). Therefore it is necessary to collect the test results using spezialiced test runners (more information coming). The results are displayed using graphs and tables, they can be grouped by responsible teams, test tool and tested artefact. So it is for example possible to exactly tell the overall test state of a tested artefact using all depending test results without manually analyse all test results of the different test tools and techniques in use.
