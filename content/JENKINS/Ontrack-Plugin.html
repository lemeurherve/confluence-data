Jenkins : Ontrack Plugin

Created by Unknown User (damien_coraboeuf), last modified on Aug 20, 2019

Plugin Information

View ontrack Jenkins on the plugin site for more information.

Older versions of this plugin may not be safe to use. Please review the following warnings before using an older version:

  • Script security sandbox bypass
  • Arbitrary code execution vulnerability
This plug-in allows to connect to an Ontrack server in order to enable traceability and monitoring of your continuous delivery pipeline(s).
