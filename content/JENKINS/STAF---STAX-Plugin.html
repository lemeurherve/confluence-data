This plugin allows Jenkins to invoke a STAF command or launch a STAX job as a build step.

About this plugin

This plugin is aimed at making it easy to launch Software Testing Automation Framework (STAF) commands and STAF eXecution engine (STAX) jobs as a build step within Jenkins.

STAF must be installed and running on the node running the build step. The STAX service must be available within your STAF environment in order to run STAX jobs.
