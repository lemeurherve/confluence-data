Jenkins : PAM Authentication Plugin

Created by Unknown User (jglick), last modified by Unknown User (jvz) on Oct 08, 2019

Adds Unix Pluggable Authentication Module (PAM) support to Jenkins.

Plugin Information

View PAM Authentication on the plugin site for more information.

Older versions of this plugin may not be safe to use. Please review the following warnings before using an older version:

  • Missing permission check allowed obtaining limited information about system configuration
  • Improper user account validation
