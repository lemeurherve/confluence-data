Jenkins : Countjobs Viewstabbar

Created by Unknown User (ialbors), last modified on Jan 11, 2012

Plugin Information

View countjobs-viewstabbar on the plugin site for more information.

This plugin shows a Views TabBar like the original one but it adds how many jobs has each view in every tab title.

Screenshots

Usage

Go to Manage Jenkins -> Configure System. Then, in the Views Tab Bar item, select CountJobs Views TabBar.

Notes

The DropDown ViewsTabBar Plugin already implements an option to show the number of jobs in each view. This one, though, sticks to the Jenkins' classic look and feel.
