Plugin Information

View OneSky on the plugin site for more information.

Maintain the localization process in sync with OneSky

Features

  • Downloading of 'ready to be published' translation files from one sky
  • Creation of the downloaded files over in your workspace
  • Pipeline support

Usage

To use this plugin 4 things need to be configured:
 

  1. The target translation project Id as found on the url of the onesky app
  2. Secret key - This is where equivalent to a password for accessing the api
  3. Public Key - This is the equivalent to a username for accessing the api
  4. Resources Path - This is the relative path, inside your workspace, where your translation files will be output to
