Plugin Information

Distribution of this plugin has been suspended due to unresolved security vulnerabilities, see below.

The current version of this plugin may not be safe to use. Please review the following warnings before use:

  • Stored XSS vulnerability

Older versions of this plugin may not be safe to use. Please review the following warnings before using an older version:

  • Persisted cross-site scripting vulnerability
  • CSRF vulnerability allowed changing graph configuration

The static analysis suite (analysis-core and the other analysis plugins) reached end-of-life. All functionality has been integrated into the Warnings Next Generation Plugin.
