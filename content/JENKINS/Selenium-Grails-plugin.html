Jenkins : Selenium Grails plugin

Created by Unknown User (kentis), last modified on Aug 23, 2009

This plugin allows you to run Selenium tests straight from a Grails applications with the selenium plugin installed.

Configuration


BaseDir          The directory where your grails application is placed relative tou your working directory.

Project name   The name of the grails application.

GrailsHome     The path to where grails is installed on the system running hudson
