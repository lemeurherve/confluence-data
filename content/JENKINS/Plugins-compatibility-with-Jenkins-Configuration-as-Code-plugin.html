Reporting issues

If you see a JCasC compatibility issues, please file a bug report with the "jcasc-compatibility" label, if one doesn't exist, to help ensure that the appropriate plugin maintainers is informed.

Once the label is added, the issues will be automatically add to the lists below. "Released as" column in the table reflect the appropriate field in JIRA, it may take a while to get the page updated.

Known issues and resolutions

See the JIRA Compatibility Dashboard

 

How to fix the issues?

In the most of the cases a plugin update is required in order to fix the issue.

Developer guidelines are under development. See the plugin documentation.
