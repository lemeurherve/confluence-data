Jenkins : Change Log History Plugin

Created by Unknown User (mindless), last modified by Unknown User (escoem) on Oct 24, 2017

Plugin Information

View Change Log History on the plugin site for more information.

This plugin copies change log data to a later build when a build is deleted.Whenever builds are deleted the change log data is lost too. If you save some builds with "keep forever" you only retain the change log of that particular build, but not other interim builds that have been deleted. This plugin watches for deleted builds and copies the change log data to the next existing build. It adds links to view the old change log data in the build page sidebar and in the changes view for projects/builds.
