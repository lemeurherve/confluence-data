Jenkins : Adaptive disconnector plugin

Created by Unknown User (olivergondza), last modified on May 01, 2014

Plugin Information

View Adaptive Disconnector on the plugin site for more information.

Put slave offline once build fails for insufficient temp or work space.

Details

Whenever build fails, rerun all node monitors on given slave in case some of the monitored metrics is over threshold. Therefore, the slave can be disconnected before next jobs are scheduled.
