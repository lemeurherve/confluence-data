Jenkins : Google Chat Plugin

Created by Unknown User (nibin), last modified on Jan 30, 2019


Send build details of a Jenkins job into google chat group


follow basic instruction to make it work

Firstly, create a google chat group unless haven't created any ,  then select  configure webhooks  option under the drop down near to the group name 

Create a new webhook URL if there is any 

Copy this URL 

Logon to  Jenkins  account 

Install google chat plugin in jenkins under manage plugins option

Go to the config page of a job 

Then select google chat notification under post build action

paste the URL in the field appeared 

Save the changes 

Every build since then will send the build details including job name , status, build number and URL of the job console will be send to the chat group specified in the job config page
