Jenkins : Pending Changes Plugin

Created by Unknown User (avogler), last modified on Feb 25, 2013

Plugin Information

View Pending Changes on the plugin site for more information.

A plugin to detect changes not part of the most recent build

Usage

The plugin adds a link to the sidepanel of projects and a column icon to indicate pending changes. There you can view all changes made in SCM after the last successful build. Currently only subversion is supported but the plugin provides extension points for additional SCM implementations. No configuration besides setting subversion options for the project is necessary.
