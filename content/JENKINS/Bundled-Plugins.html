Jenkins : Bundled Plugins

Created by Unknown User (xipmix), last modified by Unknown User (duemir) on Oct 31, 2016

To find out which plugins are bundled with Jenkins, look in the WAR file for .hpi and .jpi files.
For example (v1.630).
