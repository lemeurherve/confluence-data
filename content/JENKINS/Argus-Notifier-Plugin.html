Older versions of this plugin may not be safe to use. Please review the following warnings before using an older version:

  • CSRF vulnerability and missing permission checks allowed capturing credentials
  • Unprivileged users with Overall/Read access are able to enumerate credential IDs

Argus Notifier Plugin

The Argus Notifier plugin sends build and system metrics to an Argus server. For more configuration, demos, and more information check out the GitHub wiki.
