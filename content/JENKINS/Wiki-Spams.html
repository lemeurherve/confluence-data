Jenkins : Wiki Spams

Created by Unknown User (kohsuke), last modified by Unknown User (mc1arke) on Nov 18, 2012

To fight with Wiki spams, we are looking for volunteers to be added to the "moderators" group in Confluence that gives them the ability to remove pages, comments, attachments, and so on. If you are willing to help, please drop us a note.

Those people with the moderator access can use the following Java WebStart app to simplify the spam removal task.

  • Create ~/.jenkins-ci.org file that contains two lines "userName=YourJenkinsWikiUserId" and "password=YourJenkinsWikiPassword"
  • Launch this tool
  • Click "Scan". This process looks for recently updated pages and ranks each of them by the likeliness of the spam. Higher the score, more likely it is that it's a spam. This process will take a while.
  • Check those pages that you think are the spams
  • Click the "Clean" buton to get rid of the marked pages.

See the source code of this tool.
