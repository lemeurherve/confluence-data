Jenkins : Desktop Build Notifications - Catlight

Created by Unknown User (alexk), last modified by Unknown User (timgrayden) on Mar 07, 2017

CatLight shows build status notifications in tray. It supports multiple public and private Jenkins servers. Catlight app runs on Windows, OS X and Linux.

Use Catlight to see the current status of important builds and tests. When a build breaks down, Catlight will change color of the tray icon and show a notification. This icon will remind your team that a build needs attention.

Compared to email notifications, Catlight saves your time by focusing on the current state, and not the history of changes.

Features

Build investigations

Developer can volunteer to fix the broken build, and CatLight will notify the whole team that he is investigating that build.

Separate build history for each Git branch and pull request

Folders support

You can monitor specific jobs from folders.
