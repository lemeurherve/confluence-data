Summary

Description

A set of plugins, extending the filtering abilities of SCM plugin ones.

These filters will ignore pull requests if its title doesnt contain a sinvle valid and open JIRA ticket.

Current functionality is implemented to:

  • Bitbucket Jira Validator SCM Filter Plugin
  • GitHub Jira Validator SCM Filter Plugin

Usage

When defining a new Github Organization or Bitbucket Team/Project job, include an additional behaviour (placed under the additional separator). 

This behaviour is configurable, being mandatory to select a previously (globaly) configured JIRA Server:

In case of not selecting a JIRA server, the form won't validate:

In case of not selecting a JIRA server, the form won't validate:

Dependencies

Each filtering trait requires its related branch-source-plugin:

  • github-source-aged-refs-trait → github-branch-source-plugin & jira-plugin
  • bitbucket-source-aged-refs-trait → bitbucket-branch-source-plugin & jira-plugin
