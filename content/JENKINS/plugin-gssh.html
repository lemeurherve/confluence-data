Welcome to our CI plugin named GSSH , this plugin is developed during our Tahoe CI project for more convenient as below feature list :

Feature list:

  1. Server Group management , We can create a server group with username , password ,port . Any servers create under this server group will auto inherit from its corresponding server group , So just input server name and ip . As we known that , During our test environment set up , different project often use the uniform server port , server user name ,and password . So it can avoid us to repeat to configure the same information once and once again .
  2. Support execute remote shell during pre-builder , builder and post-builder.
