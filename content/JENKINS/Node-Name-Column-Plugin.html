Jenkins : Node Name Column Plugin

Created by Unknown User (kalpanab), last modified by Unknown User (stephenconnolly) on Oct 21, 2014

Plugin Information

View Node Name Column on the plugin site for more information.

Adds column showing node name and can be configured in views.

This simple plugin adds a Node Name selection when configuring the columns for views.
