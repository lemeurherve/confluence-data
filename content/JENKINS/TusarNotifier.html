This plugin has been deprecated. Please use the DTKit Plugin to convert outputs from the metric tools execution into unified outputs.
The DTKit Plugin provides an automatic migration procedure from this plugin.


Plugin Information

No information for the plugin 'tusarnotifier' is available. It may have been removed from distribution.

This plugin makes it possible to convert the results of an execution from a metric tool to TUSAR, and then send TUSAR outputs to Sonar.

The current supported tools are:

  • Tests tools
    • AUnit
    • BoostTest
    • C++Test
    • CppUnit
    • Junit
    • NUnit
    • PHPUnit
    • UnitTest++
  • Violations tools
    • Checkstyle
    • C++Test
    • Gnatcheck
    • Klocwork
  • Coverage
    • Cobertura
  • Measures tools
    • Gnatmetric
