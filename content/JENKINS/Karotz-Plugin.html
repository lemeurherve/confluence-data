Overview

This plugin aims to publish build results to your Karotz.

Karotz is the third generation Nabaztag, and first to be released since the Mindscape purchase. Like its predecessors, Karotz connects to the Internet using Wifi and has RFID reading capability. Additionally, it includes an integrated web cam, a USB port (which can be used for power as well as connectivity), and 256MB of onboard storage. Karotz was released in April 2011
