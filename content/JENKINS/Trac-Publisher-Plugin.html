Jenkins : Trac Publisher Plugin

Created by Unknown User (batkinson), last modified by Unknown User (lshatzer) on Aug 06, 2013

This plug-in adds a publisher that posts build links to Trac tickets referenced in built scm revisions.

Plugin Information

View Trac Publisher on the plugin site for more information.

The current version of this plugin may not be safe to use. Please review the following warnings before use:

  • Credentials stored in plain text
