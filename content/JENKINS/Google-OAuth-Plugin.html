Jenkins : Google OAuth Plugin

Created by Unknown User (mattmoor), last modified by Unknown User (craigbarber) on Mar 08, 2019

This plugin allows for the registration of Google Service Account Credentials with the Jenkins master, which can be used to interact with Google APIs.

Plugin Information

View Google OAuth Credentials on the plugin site for more information.

Older versions of this plugin may not be safe to use. Please review the following warnings before using an older version:

  • Arbitrary file read vulnerability

Consult the google-oauth-plugin repository on GitHub for documentation.
