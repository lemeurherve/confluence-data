No information for the plugin 'kerberos-authenticator' is available. It may have been removed from distribution.

The Kerberos Authenticator Plugin allow use a Kerberos server as Jenkins Securty Authentication.

The plugin will require the KDC server and the Realm to authenticate with.

Standard Kerberos Configuration will be created by the Plugin at the JENKINS_HOME folder (krb5.conf and jaas.conf), the user can overwrite those files with a new configuration,
