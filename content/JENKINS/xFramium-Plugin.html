Plugin Information

View XFramium Builder on the plugin site for more information.

 

The xFramium Platform plugin allows users to execute xFramium scripts inside of Jenkins and view reports.  Users can simply specify and validate their driver configuration file

Or they can modify the execute parameters to override the suite name, specify test cases and tags or specify devices and clouds.

 

View the xFramium reports directly from Jenkins
