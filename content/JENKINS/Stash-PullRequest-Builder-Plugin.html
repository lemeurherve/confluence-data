Jenkins : Stash PullRequest Builder Plugin

Created by Unknown User (nemccarthy), last modified by Unknown User (proski) on Sep 02, 2019

Plugin Information

View Stash Pullrequest Builder on the plugin site for more information.

This Jenkins plugin builds pull requests from an Atlassian Bitbucket Server (formerly known as Stash) and reports the build result as a comment on the pull request page. Builds are normally triggered by creating or updating to the pull request, but it's also possible to trigger a build with a comment.

Prerequisites

  • Jenkins 2.60.3 or higher
  • Git Plugin

Recommended

  • Bitbucket (Stash) Notifier Plugin to notify Bitbucket Server of the build results in a more computer-friendly way, so you can require a successful build before the pull request can be merged
  • OWASP Markup Formatter Plugin ("Safe HTML" formatter) to show links to the Bitbucket Server in the build history (go to Manage Jenkins -> Configure Global Security and enable Safe HTML markup)
