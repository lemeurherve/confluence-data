Jenkins : Keep slave offline

Created by Unknown User (vjuranek), last modified by Unknown User (lshatzer) on Aug 06, 2013

Plugin Information

View keepSlaveOffline on the plugin site for more information.


  This plugin allows to keep given slave offline after Jenkins restart.


This is a workaround for a bug in pre-1.397, 1.397+ don't need it. Typical usage is take Jenkins slave offline for some run running job over the weekend and make sure that slave stay offline even if there is maintenance Jenkins restart over the weekend.
