With 7 jobs and 10 threads running on Xenu link checker, there were multiple "not found" messages reported for links in Xenu link checker, along with reports of "temporarily overloaded" and one of the Jenkins jobs was performing build 3 when the build seems to have disappeared.  1 and 2 are visible and browsable, 4 is pending, but 3 is not in the list.  The 3 directory is on the disc, but only inluces a changelog.xml directory and a log directory.

Old JDK test

Run Jenkins with Debian Squeeze default JDK (JDK 1.6.0_20)

Performs as expected

JDK 1.6.0_20 is not well behaved with Jenkins.  A newer JDK 6 is needed.  Since the JDK 6 provided with Debian Squeeze is not being updated, it is probably best to switch to the Oracle JDK 6 or Oracle JDK 7 on Debian Squeeze.  A PPA installer for Oracle JDK 7 is available from launchpad and works quite well with Debian Squeeze.

1.480.1 / WARN
