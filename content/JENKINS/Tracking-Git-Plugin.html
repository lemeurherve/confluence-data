Jenkins : Tracking Git Plugin

Created by Unknown User (deanaldinger), last modified on Feb 18, 2014

Plugin Information

View Tracking Git on the plugin site for more information.

Lets one project track the Git revisions that are built for another project.


When building, the TRACKING_GIT_BUILD environment variable will contain the URL of the tracked build.

It is generally useless to use SCM polling if Git tracking is enabled. Instead, trigger this project from the tracked project.
