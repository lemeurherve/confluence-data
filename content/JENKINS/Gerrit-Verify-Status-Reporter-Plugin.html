Jenkins : Gerrit Verify Status Reporter Plugin

Created by Unknown User (zaro0508), last modified on Oct 06, 2016

Plugin Information

View Gerrit Verify Status Reporter on the plugin site for more information.

Allows Jenkins jobs to send test results to Gerrit instances that use the verify-status plugin

Summary

This plugin is used used to support the Gerrit verify status plugin.  You should probably read about the Gerrit plugin before trying to use this plugin.

There is a quick start guide for this plugin is in the GitHub repository
