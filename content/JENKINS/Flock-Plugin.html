Jenkins : Flock Plugin

Created by Unknown User (ajaymann), last modified on Aug 07, 2019

Plugin Information

View Flock Notification on the plugin site for more information.

This plugin will provide build status notifications to Flock messenger. 

Installation

  1. Go to 'Manage Jenkins' from the Jenkins homepage.
  2. Navigate to Manage Plugins
  3. Change the tab to Available
  4. Search for Flock
  5. Check the box next to install.
