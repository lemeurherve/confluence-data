Jenkins : Gallio Plugin

Created by Anonymous, last modified by Unknown User (gbois) on Aug 14, 2014

Plugin Information

View Gallio on the plugin site for more information.

This plugin makes it possible to publish Gallio/MbUnit test results.

Gallio/Mbunit reports are imported from each build into Jenkins so they are displayed with a trend graph and details about which tests that failed.
