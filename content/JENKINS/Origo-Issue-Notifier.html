This plugin opens or updates an issue on an Origo instance based on the build result.

Changelog

Version 1.1 (May 18, 2010)

  • Initial release

Usage

After installing this plugin, in the configuration of a job under Post-build Actions you can select "Origo issue notifier" and enter the configuration parameters. After a newly failed build a new issue will be created. After a newly recovered build the issue will be closed.
