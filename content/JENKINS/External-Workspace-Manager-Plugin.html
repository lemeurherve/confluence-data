Jenkins : External Workspace Manager Plugin

Created by Unknown User (alexsomai), last modified by Unknown User (deepchip) on Aug 27, 2016

Plugin Information

View External Workspace Manager on the plugin site for more information.

This plugin provides an external workspace management system. It facilitates workspace share and reuse across multiple Jenkins jobs. It eliminates the need to copy, archive or move files.

Documentation

The plugin documentation may be found on the GitHub README page.

This plugin was developed under the Google Summer of Code 2016 program. The GSoC final presentation for this plugin can be found on youtube, and the slides are on-line as well.
