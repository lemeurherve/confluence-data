Created by Unknown User (teilo), last modified by Unknown User (lshatzer) on Mar 03, 2016

About

The CloudBees Docker Events Hook is a simple server that will interrogate a Docker server for container and image related events and notify the CloudBees Docker Traceability plugin, allowing your container deployments to be linked back through to the build that created them.

This server is currently work in progress.

Source Code

The source code for this can be found in the docker-events-hook repository on GitHub
