Plugin Information

No information for the plugin 'status-view' is available. It may have been removed from distribution.

View type to show jobs filtered by the status of the last completed build.

This plugin adds a new view type that can be selected when adding job views. It works like the normal List View, but filters the selected jobs by the status of the last completed build. It may also be used to show only jobs that are currently building.
