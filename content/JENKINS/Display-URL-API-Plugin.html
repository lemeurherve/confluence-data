Jenkins : Display URL API Plugin

Created by Unknown User (jamesdumay), last modified by Unknown User (olamy) on Aug 03, 2019

Plugin Information

View Display URL API on the plugin site for more information.

Provides an extension point to define alternate URLs for use in notifications.

URLs can be requested/extended for these UI locations:

  • Root page.
  • Job.
  • Run.
  • Run changes.
  • Test result.
