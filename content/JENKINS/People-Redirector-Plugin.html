Jenkins : People Redirector Plugin

Created by Unknown User (m2spring), last modified on May 12, 2012

Plugin Information

View People Redirector on the plugin site for more information.

Allows you to integrate with an external user management system.

HTTP requests which go to the Jenkins-internal "people" page are redirected to the external site.
