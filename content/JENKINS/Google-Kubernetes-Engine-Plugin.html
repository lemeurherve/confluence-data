Jenkins : Google Kubernetes Engine Plugin

Created by Unknown User (craigbarber), last modified on Mar 08, 2019

The Google Kubernetes Engine (GKE) Plugin allows you to publish deployments built within Jenkins to your Kubernetes clusters running within GKE.

Plugin Information

View Google Kubernetes Engine on the plugin site for more information.

Older versions of this plugin may not be safe to use. Please review the following warnings before using an older version:

  • RCE vulnerability
  • Missing permission checks allowed obtaining limited information about a credential scope
  • Temporary secret stored in a user accessible location
