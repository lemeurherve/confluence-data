Jenkins : JSWidgets Plugin

Created by Unknown User (mfriedenhagen), last modified by Unknown User (lshatzer) on Aug 09, 2013

Allows embedding various statistics available from Jenkins in your via javascript-snippets ala ohloh.net.

Plugin Information

View Javascript Widgets on the plugin site for more information.

With this plugin, you can integrate the output of some Jenkins statistics to external sites by including either a javascript snippet,
an open social gadget or using an iframe.

For examples follow the JS Widgets link at http://huschteguzzel.de/hudson/job/jenkinsci-jswidgets-plugin/jswidgets/
