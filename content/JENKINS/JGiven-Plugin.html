Jenkins : JGiven Plugin

Created by Unknown User (wolfs), last modified on Dec 26, 2015

Plugin Information

View JGiven on the plugin site for more information.

This plugin lets you generate JGiven reports as a post-build action.

Job DSL

The current supported dsl is as follows:

freeStyleJob('test-job') {
  publishers {
    jgivenReports {
      excludeEmptyScenarios() // Since 0.10
      html {
        customCss 'some.css'
        customJs 'some.js'
        title 'My Custom Title'
      }
      results '**/*.json'
    }
  }
}
