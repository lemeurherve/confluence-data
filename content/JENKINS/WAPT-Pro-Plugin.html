Jenkins : WAPT Pro Plugin

Created by Unknown User (waptpro), last modified on Sep 12, 2017

Plugin Information

View WAPT Pro on the plugin site for more information.

This plugin lets you run load tests for web application projects with help of WAPT Pro. During the test you can monitor the process through the Jenkins console. Test results are combined with other integration jobs in the dashboard.
