Jenkins : Aqua Security Scanner Plugin

Created by Unknown User (norbyltd), last modified by Unknown User (oranmoshai) on Sep 19, 2019

Plugin Information

View Aqua Security Scanner on the plugin site for more information.

Older versions of this plugin may not be safe to use. Please review the following warnings before using an older version:

  • Plain text password shown in configuration form
  • Aqua Security Scanner Plugin stores credentials in plain text

Description

Adds a Build Step for scanning Docker images, local or hosted on registries, for security vulnerabilities, using the API provided by Aqua Security.
