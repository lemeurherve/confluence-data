Jenkins : Hudson Monitor Gadget for Microsoft Sidebar.

Created by Unknown User (jm.taillant), last modified on Jan 11, 2011

This Hudson Monitor gadget for Vista / Windows 7 sidebar provides an easy way to keep an eye on the status of your Hudson jobs.

Work with Hudson v1.92 and later

Installation

Download the gadget package file from http://code.google.com/p/hudsonmonitor/

On a windows system, simply double-click the downloaded file to have it register itself with Microsft Gadgets. Use Gadget settings to get jobs from Hudson servers.
