Jenkins Settings: ZAP Proxy Settings

Jenkins Manage Jenkins Configure System ZAP

Configure the default ZAP host (e.g. 127.0.0.1) and the default ZAP port (e.g. 9090). They can be overridden for each job.

The host and port set here should be the SAME set in Firefox and in the ZAP security tool.

Notice: This should be the IP address of the Slave (the machine where ZAP security tool is installed).

Notice: Depending on your Jenkins version, you will need to press Save at least once in order for these settings to persist into a Job's configuration.
