Jenkins : Plain Credentials Plugin

Created by Unknown User (jglick), last modified on Dec 11, 2018

Allows use of plain strings and files as credentials.

Plugin Information

View Plain Credentials on the plugin site for more information.

Description

This plugin provides two new types of credentials:

  • FileCredentials or Secret file:

  • StringCredentials or Secret text:

Usage

These credentials were originally designed to be used together with the Credentials Binding Plugin. See the wiki for an example use case.

However, as described in the Credentials Plugin wiki, they could potentially be used by any other plugin that requires them.
