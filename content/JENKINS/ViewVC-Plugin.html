Jenkins : ViewVC Plugin

Created by Unknown User (mindless), last modified by Unknown User (mirumpf) on Jan 24, 2014

Plugin Information

View ViewVC on the plugin site for more information.

This plugin integrates ViewVC browser interface for CVS and Subversion with Hudson.
Entries in change logs will by hyperlinked to the specified files/diffs in ViewVC.
