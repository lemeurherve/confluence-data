discoBit

discoBit is a central configuration repository for applications. Properties are read from the repository during bootstrap or on demand by your software.

See http://discobit.com for details.

Autoconfig Plugin

This plugin pushes selected property files from the Jenkins workspace to the discoBit configuration repository.
