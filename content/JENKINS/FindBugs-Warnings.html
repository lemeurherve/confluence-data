FindBugs Warnings

On the warnings per package view, the FindBugs plug-in lists all warnings in a Java package. Each warning is shown with the corresponding warning message and a detailed warning description from the FindBugs team. If you are viewing the results of the latest build then there are links available to the corresponding source file.
