Jenkins : Dot Jenkins Ci Dot Org

Created by Unknown User (kohsuke), last modified on Nov 10, 2011

Some of our tools use ~/.jenkins-ci.org to retrieve your jenkins-ci.org account credential. This file follows a Java properrty file format, which is basically as follows:

userName=kohsuke
password=mypassword

any line that starts from '#' is treated as comments.
