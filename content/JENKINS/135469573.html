Jenkins : Chinese (Simplified) Translation

Created by Unknown User (nodexy), last modified by Unknown User (surenpi) on Sep 28, 2018

English:

You are welcome to contribute in translating moe Jenkins stuff, or suggesting changes to the actual translation.

'translation-tool' is a useful piece of code created to do the translation, it could help you in the case you want to contribute to translate anything.

Chinese (Simplified):

欢迎参与Jenkins 简体中文翻译项目。请从了解和学习翻译工具 'translation-tool' 开始。


规范

本地化翻译请遵循规范，如对改规范有疑问，请在中文本地化兴趣组中讨论。

贡献

根据 JEP-216 的提议，Jenkins 核心及其插件的中文本地化资源文件需要放置在简体中文插件中。
