Jenkins : Artifact Manager S3 Plugin

Created by Unknown User (jglick), last modified on May 23, 2019

Plugin Information

View Artifact Manager on S3 on the plugin site for more information.

A Jenkins plugin to keep artifacts and Pipeline stashes in Amazon S3.

  • Extend 1.5’s flush fix to some error-handling cases.
