Jenkins : Runscope Plugin

Created by Unknown User (mansilladev), last modified by Unknown User (frankstratton) on Apr 25, 2016

Plugin Information

View Runscope on the plugin site for more information.

This plugin allows you to add a Runscope API test as a build step into your Jenkins build pipeline. The plugin will trigger a specific API test (via Trigger URL) and wait for the test to complete. If the test passes, the build steps will continue. However, if it fails, the build will be marked as failed.
