Jenkins : Text Finder Run Condition Plugin

Created by Unknown User (cjo9900), last modified on Jul 16, 2012

Plugin Information

View Text Finder Run Condition on the plugin site for more information.

Features

This plugin implements a text finder condition to be used by the Run Condition Plugin.
This is a condition based on the Text-finder Plugin with some minor changes.

  • It always returns true on the first occurrence of the match,
  • The console log is checked before the files, if enabled.
  • If no files are found returns False.
