Column showing the last success description that can be configured in views.Adds a column showing the build description of the last successful build that can be configured in views.

This simple plugin adds a Last Success Description selection when configuring the columns for views.

To set the build description automatically use http://wiki.jenkins-ci.org/display/HUDSON/Description+Setter+Plugin
