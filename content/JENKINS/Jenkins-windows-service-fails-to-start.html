Jenkins : Jenkins windows service fails to start

Created by Unknown User (evernat), last modified by Unknown User (ido_ran) on Feb 19, 2013

Service Fails to Start

Symptoms

* Event Viewer, App: "Service cannot be started. The service process could not connect to the service controller"
* No log files in Jenkins directory
** Repeated attempts to start server may result in 'jenkins.err.log' and 'jenkins.out.log'
* jenkins.war file is zero length

Solution

* Copy original 'jenkins.war' into 'jenkins' directory and restart service
