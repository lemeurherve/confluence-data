View RadarGun on the plugin site for more information.

Older versions of this plugin may not be safe to use. Please review the following warnings before using an older version:

  • RCE vulnerability
This plugin allows to run RadarGun benchmark as a Jenkins build step.
RadarGun is a data grid and distributed cache benchmarking framework built to test Infinispan and other distributed data grid platforms.

Configuration and usage manual is available on GitHub.
