Displays a picture of Bruce Schneier (instead of Jenkins the butler) and a random Bruce Schneier fact on each build page.


Installation

Install from Jenkins Plugins Update Center.

Usage

  1. After installing the plugin, a checkbox with the text 'Active BruceSchneier' should appear on a configuration page of a job.
  2. Tick the checkbox and Bruce Schneier image (along with the random quote) will appear on the job page and the future build pages.
