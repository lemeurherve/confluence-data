Plugin Information

No information for the plugin 'gant' is available. It may have been removed from distribution.

This plugin allows Hudson to invoke Gant build script as the main build step.

The plugin works much like Ant builder, and using it is very easy. You basically just need to specify the Gant targets that you want to run:

If 'gant' is not in PATH, or if you have multiple Gant installations and need to be able to configure which project uses which Gant, then you can go to the system configuration and tell Hudson where your Gants are:

As of 0.3, Gant doesn't report the build status by its exit code, so to keep track of a build failure, you'll need to use Text-finder Plugin to recognize a build failure.
