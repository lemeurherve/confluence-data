Jenkins : vCommander+Plugin

Created by Unknown User (embotics) on May 29, 2018 
{jenkins-plugin-info:embotics-vcommander-plugin}

This Jenkins plugin is designed to work with Embotics' vCommander cloud orchestration and automation software.  It allows Embotics vCommander users to create jobs in Jenkins and connect to vCommander through REST API v.3 connections.  With this integration you can submit service requests, get status on existing requests and apply command workflows to Embotics-managed virtual machines in any supported cloud platform.

Embotics vCommander multi-cloud management platform is the fastest and easiest way to automate provisioning, enforce governance, and enable self-service IT across virtualized, private and public cloud infrastructures.
