Jenkins : JUnit Realtime Test Reporter Plugin

Created by Unknown User (olivergondza), last modified by Unknown User (jglick) on Sep 02, 2017

Plugin Information

View JUnit Realtime Test Reporter on the plugin site for more information.

Report test result while the build is still running.

Usage

Tick "Visualize test results in real time" in job configuration to turn this feature on. Builds with Junit publisher and maven builds will have an action called "Realtime Test Result" while in progress.
