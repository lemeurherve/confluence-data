Jenkins : Scala JUnit Name Decoder Plugin

Created by Unknown User (dnadolny), last modified on May 23, 2013

Plugin Information

View Scala JUnit Name Decoder on the plugin site for more information.

This plugin displays JUnit test names for Scala classes as they appear in source, eg "some$u0020test" becomes "some test".
