Jenkins : Micro Focus Deployment Automation Plugin

Created by Unknown User (akevinlee), last modified on Feb 13, 2017

Plugin Information

No information for the plugin 'microfocus-da' is available. It may have been removed from distribution.

This plugin allows Micro Focus Deployment Automation to be used in Jenkins Pipelines

Features

The plugin currently provides the following adhoc build steps:

  • Create Version
  • Add Files to Version
  • Add Status to Version
  • Run Application Process

The steps can be used in "freestyle" projects but are designed to be used in pipelines.

Requirements

This plugin is designed for Micro Focus Deployment Automation 6.1.3 or later.
