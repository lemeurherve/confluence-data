Jenkins : Webhook Relay Plugin

Created by Unknown User (bwalding), last modified on Feb 26, 2017

This plugin is not currently OSS as of February 2017 - but we are currently moving all the pieces into place for an OSS release.

The Webhook Relay Plugin publishes hooks (from an external webhook based system (e.g. Github / Dockerhub)) to subscribers on an internal network. This is primarily used for secure systems which do not expose their Jenkins instance to the outside world.
