Jenkins : Spanish Translation

Created by Unknown User (manuel_carrasco), last modified on Jul 22, 2010

Translation to Spanish is handled by Manuel Carrasco Moñino.

You are welcome to contribute in translating more Hudson stuff, or suggesting changes to the actual translation, if so, please feel free to get in touch with Manuel.

'translation-tool ' is a useful piece of code created to do the Spanish translation, it could help you in the case you want to contribute to translate anything.

Discussions

The terminology used to translate Hudson elements into Spanish can be controversial, we would like to know your opinion. If you have any suggestion leave your comments in the bottom of this page.
