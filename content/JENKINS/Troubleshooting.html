Jenkins : Troubleshooting

Created by Unknown User (kinow), last modified on Nov 20, 2014

When filing issues, in order to help us and to speed up our feedback, the following tips apply:

  • Take a look at Log generated by the plug-in
  • Look at the Chrome JS console
  • Report as much information as possible and, if possible, include screen shots and XML configurations
  • Take care with sensitive information being included in these artefacts

Uno Choice Logs

Create a new logger for the plug-in, using org.biouno.unochoice as Logger value. This will create a new Logger that will contain more detailed logging information.
