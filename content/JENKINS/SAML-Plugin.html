Plugin Information

View SAML on the plugin site for more information.

Older versions of this plugin may not be safe to use. Please review the following warnings before using an older version:

  • CSRF protection for any URL can be bypassed
  • Keystore and private key passwords stored unencrypted
  • HTTP session fixation vulnerability
This plugin allows authentication to Jenkins via the SAML 2.0 protocol.
