Jenkins : Selenium Builder Plugin

Created by Unknown User (rossrowe), last modified on Jul 12, 2013

Plugin Information

View Selenium Builder on the plugin site for more information.


Invokes Selenium Builder scripts from a Jenkins build

Configuration

Once the plugin has been installed, it can be configured via the Job Configuration page. Click the 'Add Build Step' button and select 'Invoke Selenium Builder script'.

Enter the location of the Selenium Builder JSON file that you wish to be invoked by your build.
