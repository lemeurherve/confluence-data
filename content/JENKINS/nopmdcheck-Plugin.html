Jenkins : nopmdcheck Plugin

Created by Unknown User (quramy), last modified on Oct 28, 2012

Plugin Information

View NOPMD check on the plugin site for more information.

This plugin checks the keyword 'NOPMD' in your codes.

'NOPMD' is a evil word to throw off PMD's checking(detail: http://pmd.sourceforge.net/suppressing.html#NOPMD).
If using this plugin with PMD Plugin, you can 'double-check' sources.
