Jenkins : Pom2Config Plugin

Created by Unknown User (mklein), last modified on Dec 13, 2013

Jenkins plugin which allows to transfer settings from a POM to a job configuration.

Plugin Information

View pom2config on the plugin site for more information.

The plugin parses a given POM and gives you afterwards the opportunity to replace different job settings like project description or the SCM URL with the found values from the POM.

The plugin shows up in the left bar of a job. There are three possibilities to give the plugin a POM file to parse, you can either use an existing POM in the workspace of the current job, get it from an URL or just upload one.

The following functions are supported

  • Project description
  • Editable Email Notification (recipient list only)
  • Git SCM URL
  • SVN SCM URL
