Jenkins : Remove all disabled modules in Maven jobs

Created by Unknown User (aheritier), last modified by Unknown User (schristou) on Jan 20, 2017

To remove all disabled modules in Maven jobs

Jenkins.instance.getAllItems(hudson.maven.MavenModuleSet.class).findAll{job -> job.isBuildable()}.each{
job ->
  job.getDisabledModules(true).each{module -> module.delete()} 
  println(job.name+" cleaned")
}
