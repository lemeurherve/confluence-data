Jenkins : Mail address resolution

Created by Unknown User (olivergondza), last modified on Apr 11, 2013

Every plugin can contribute to mail address resolution process using hudson.tasks.MailAddressResolver extension point.

Performance considerations

Mail address resolution is a potentially time consuming process. It can considerably slow down operation relaying on mail address resolution, especially on Jenkins instances having registered several resolvers.
