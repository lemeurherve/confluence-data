Jenkins : SVNCompat13 Plugin

Created by Unknown User (mindless), last modified by Unknown User (lshatzer) on Aug 05, 2013

This plugin forces the built-in SVNKit library to use the Subversion 1.3 working copy format (instead of the most recent one it supports).

Plugin Information

View SVN 1.3 Compatibility on the plugin site for more information.
