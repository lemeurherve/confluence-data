Jenkins : Distinguishable Gray Balls Plugin

Created by Unknown User (rstone), last modified on Apr 23, 2015

Plugin Information

No information for the plugin 'distinguishable-gray-balls-plugin' is available. It may have been removed from distribution.

Adds different icons for Disabled and Aborted jobs because the default Jenkins icons don't differentiate between Not Built, Disabled and Aborted.

Forked from the Green Balls plug-in

This plug-in is compatible with both blue and green ball icons. If you want a more modern looking icons consider using the Modern Status Plugin instead.
