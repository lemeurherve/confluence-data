Jenkins : aRESTocats-Plugin

Created by Unknown User (qacwnfq), last modified on Jun 10, 2018

Plugin Information

View aRESTocats on the plugin site for more information.

Jenkins-Plugin for tracking metrics created with the aRESTocats REST API testing toolkit available at https://www.npmjs.com/package/arestocats and https://github.com/aixigo/arestocats.

Issues:
