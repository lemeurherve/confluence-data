Jenkins : Agile Cockpit Notification Plugin

Created by Unknown User (prakashan_prowareness), last modified on Mar 02, 2016

Summary

This plugin send the build information to cockpit application. When the Jenkins job is configured with this plugin, the build information is sent to URL as mentioned in the plugin.

Configuration

  • Once the plugin is installed, you can the Agile-Cockpit-Notifications section as shown below.

  • Click on Add Endpoint and configure the fields.
