The Hall plugin publishes build status to your room at Hall.com

Usage

You will need to get a token for the room you want to post to at https://hall.com/docs/integrations/jenkins. This token can be used for as many builds as you'd like.

  1. Once installed, open a job.
  2. Under Post-build actions, select Add post-build action and choose Publish to Hall
  3. Enter your Room API token, found at https://hall.com/docs/integrations/jenkins.
  4. Select whether you'd like to be notified on Success, Failure and/or Unstable
  5. You're Done! Jenkins' alerts will now post to your room in real time.
