Jenkins : Anonymous Uptime Check Plugin

Created by Unknown User (jknurek) on Oct 30, 2018 
{jenkins-plugin-info:anonymous-uptime-check}

Description

Allows unauthenticated access to check Jenkins' ability to run new jobs at /anonymousUptimeCheck.

If Jenkins is quieting down, the response is a 503, otherwise a success 200 json repsonse.


This can be used for external tools (like pingdom, uptrends, pagerduty, etc.)Â to check track uptime statics of Jenkins instance.
