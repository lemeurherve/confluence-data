Jenkins : Signal killer

Created by Unknown User (vjuranek), last modified by Unknown User (victor) on Jan 09, 2015

Plugin Information

View signal-killer on the plugin site for more information.

Note: GitHub link is https://github.com/jenkinsci/signal-killer

  This plugin extends ProcessKiller extension point and sends signal (currentlly SIGKILL) to given job via native GNU C library call.   


Currently there's no possibility how to configure which signal should be sent.
